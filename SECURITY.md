# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 2.x     | :white_check_mark: |
| < 2.0   | :x:                |

## Reporting a Vulnerability

NoSQLHunter is a **security testing tool** designed for authorized penetration testing
and vulnerability assessment. If you discover a vulnerability in NoSQLHunter itself,
please report it responsibly.

**Do not** open public issues for security vulnerabilities. Instead,
send a detailed report to the repository owner via GitHub.

### What to include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if applicable)

## Responsible Disclosure

We request that you:
1. Notify us immediately upon discovering a vulnerability
2. Allow reasonable time for a fix before public disclosure
3. Do not use the vulnerability maliciously

## Legal Notice

This tool is intended for **authorized security testing only**. Users are
responsible for complying with applicable laws and obtaining proper
authorization before testing any systems.
